## 三、病人管理

### 3-1 新增病人
1. 點左側「病人管理」
2. 右上角「新增病人」按鈕
3. 填寫基本資料：姓名、病歷號、性別、出生日期、癌別
4. 點「儲存」

### 3-2 查詢病人
- 上方搜尋列輸入**姓名**或**病歷號**即時篩選
- 可依癌別、狀態（在案/結案/已歿）篩選
- 多重癌病人（同病歷號有多筆記錄）會顯示**「多重癌」**紅色標籤

### 3-3 病人詳情
點擊任一病人進入詳情頁，共 7 個分頁：

| 分頁 | 內容 |
|------|------|
| 總覽 | 基本資訊摘要、最近事件 |
| 基本資料 | 個人基本資訊編輯 |
| 分期 | 臨床/病理分期（cTNM/pTNM） |
| 收案/癌別 | 收案資訊、癌別特有欄位 |
| 事件記錄 | 所有治療事件（手術/化療/放療等）的新增與管理 |
| 指標 | 這位病人相關的品質指標達成狀況 |
| 追蹤 | 追蹤計畫與回診記錄 |

### 3-4 新增事件
1. 在病人詳情頁點「事件記錄」分頁
2. 右上角「新增事件」
3. 選擇事件類型（手術、化療、放療、追蹤等）
4. 填寫相關欄位後儲存

> **MDT 相關事件**：若某事件需要提交 MDT 討論，勾選「需要 MDT 討論」flag，建立 MDT 會議時系統會自動偵測並列入必要提報名單。

## 四、多專科會議（MDT）

### 4-1 行事曆
- **實色徽章**：已建立的會議，點擊查看詳情
- **虛線徽章**：系統根據排程原則自動推算的建議日期，點擊直接建立會議
- **統計 tab**：各癌別群組討論量、每月趨勢、必要提報類別分布

### 4-2 建立會議
1. 點行事曆上的虛線日期，或右上角「新建會議」
2. 選擇**癌別群組**（乳癌/胸腔癌/頭頸癌…）
3. 系統自動帶入：建議排程原則、負責個管師、會議地點、預設時間
4. 日期/時間/地點皆可手動修改（實際借到的場地可能不同）
5. 點「建立會議」

### 4-3 複製會議
在已建立的會議詳情面板：
1. 點「**複製**」按鈕
2. 選擇目標日期（通常是下個月同一癌別的日期）
3. 確認複製：病人清單會複製過去，討論要點/結論清空讓個管師重新填寫

### 4-4 進入會議室
在會議詳情面板點「**進入會議 →**」，進入會議室有 5 個區塊：

| Section | 說明 | 自動來源 |
|---------|------|---------|
| ⭐ 新個案討論 | 上次同癌別 MDT 後新收的病人 | 系統自動列出，個管師勾選 |
| 📈 前期追蹤 | 上次 MDT 有結論、這次要回報 | 系統自動列出 |
| ⚠️ 必要提報 | 符合基準2.4八類條件的病人 | 系統自動偵測（⚡標示） |
| 📋 醫療小組 | 個管師回覆意見追蹤 | 有醫療小組事件卡的病人 |
| 👤 特殊議程 | 臨時加入 | 手動搜尋加入 |

**手動加入病人：**
- 點各 Section 右上的「加入病人」
- 輸入姓名或病歷號搜尋
- 選擇要加入哪個 Section
- 必要提報/醫療小組類型會同步幫病人建立事件卡

**填寫討論內容：**
- 點擊病人卡片展開
- 自動帶入治療記錄（手術/化療/放療）
- 填寫：影像結果、病理結果、討論要點、**結論與建議**（有結論邊框轉綠）、後續追蹤計畫

### 4-5 會後填寫
點 Header「**會後填寫**」按鈕：
- 一次看所有討論個案，補填摘要/結論/追蹤
- **帶入下次追蹤 toggle**：關閉後，此個案下次不會自動出現在前期追蹤名單
- 儲存時自動更新病人的 MDT 事件卡結論

### 4-6 完成會議
點「**完成會議**」：
1. 所有討論內容自動儲存
2. 有結論的病人自動新增 `MDT 結論` 事件卡
3. 已標記 `需要 MDT 討論` 的事件卡自動標記為已討論
4. 會議狀態改為「已完成」

### 4-7 產出功能
Header 右側有四個產出按鈕，前方五顆色票可選配色主題：

| 按鈕 | 說明 |
|------|------|
| **HTML** | 下載 HTML 投影片，瀏覽器直接開啟播放 |
| **LINE** | 產生會議通知文字，自動複製到剪貼簿 |
| **DOCX** | 下載 Word 格式會議記錄 |
| **PPTX** | 下載 PowerPoint 簡報 |

**HTML 投影片操作：**
- ← → 或空白鍵翻頁
- `F` 全螢幕
- `Home` / `End` 跳到第一/最後一頁

**配色主題（5種）：**
彰濱經典（預設）/ 暖陽 / 森林 / 薰衣草 / 高對比（列印友善）
> 點選後立即生效，下次進入維持上次選擇

## 五、指標管理

### 5-1 待補項目
- 列出哪些病人有哪些指標欄位尚未填寫
- 點擊病人→直接跳到對應的詳情分頁補填

### 5-2 指標總覽
- 所有癌別的60項指標達成率
- 可依年度、癌別篩選

### 5-3 趨勢預警
- 低於目標值的指標以視覺化方式顯示

### 5-4 資料匯出
- 可預覽後下載 Excel 格式的指標申報資料

## 六、工作中心

自動偵測需要追蹤的病人，依緊急度分為三區：

| 區域 | 觸發條件 |
|------|---------|
| 🔴 緊急 | 事件逾期未完成超過 7 天 |
| 🟡 本週 | 本週到期或本週有排程事件 |
| ⚪ 需關注 | 長期未訪視、新收無治療計畫等 |

點擊任一項目→直接跳到對應病人的詳情頁。

## 八、系統工具

### 重置資料庫
> ⚠️ **刪除所有資料並重新載入預設資料，不可復原**

1. 系統工具 → 資料管理
2. 點「重置資料庫」
3. 輸入驗證碼確認
4. 重置成功後頁面自動重新載入

### 科別/醫師同步
換版後如果發現醫師選單缺少新科別：
1. 系統工具 → 資料管理
2. 點「**立即同步科別/醫師**」
3. 系統自動補入缺少的科別和醫師（不刪除現有資料）

### 刪除測試資料
病歷號以 `T-` 開頭的測試病人及相關 MDT 會議記錄。

## 九、常見問題

**Q：換版後登入密碼錯誤？**
A：換上新版後重新整理頁面，系統會自動補建新帳號（`14002`/`11750`）。

**Q：多專科會議頁面空白，沒有測試會議？**
A：系統工具 → 重置資料庫，重置後會載入3場測試 MDT 會議。

**Q：科別選單沒有影像診斷科/病理科？**
A：系統工具 → 立即同步科別/醫師。

**Q：HTML 投影片顯示系統字體，不是細明體？**
A：HTML 投影片需要網路連線才能載入 Google Fonts（Noto Sans TC），若在無網路環境使用，會自動 fallback 到系統預設字體，功能正常。
