# 彰濱癌症中心 - 個管病患追蹤系統

基於 React + TypeScript + IndexedDB 的個管病患追蹤系統，可部署到 GitHub Pages。

**當前版本：V4.6**

## ⚠️ 重要警告

1. **資料存在瀏覽器**：清除瀏覽器資料會遺失所有病歷！
2. **iOS 7 天清除**：iOS Safari 會在 7 天未使用後自動清除網站資料
3. **必須定期備份**：將備份檔儲存到 iCloud / Google Drive / 電腦

## 🚀 下載更新檔後的測試步驟

### 方式一：使用 npm（推薦）

```bash
# 1. 解壓縮 ZIP 檔到任意位置

# 2. 開啟終端機，進入專案資料夾
cd /path/to/case-manager-web

# 3. 安裝依賴（只需第一次或更新後執行）
npm install

# 4. 啟動開發伺服器
npm run dev

# 5. 瀏覽器會自動開啟，或手動訪問顯示的網址（通常是 http://localhost:5173）
```

### 方式二：使用 Python 簡易伺服器

```bash
# 1. 解壓縮 ZIP 檔

# 2. 進入專案資料夾
cd /path/to/case-manager-web

# 3. 啟動伺服器
# Mac/Linux:
cd dist && python3 -m http.server 8080

# Windows:
cd dist && python -m http.server 8080

# 4. 開啟瀏覽器訪問 http://localhost:8080
```

### 方式三：直接使用建置好的檔案（需伺服器）

專案內的 `dist/` 資料夾是已建置好的靜態檔案，可直接部署到任何靜態網站伺服器。

## 📱 手機測試

1. 確保手機和電腦在同一個 WiFi 網路
2. 在電腦執行 `npm run dev -- --host`
3. 終端機會顯示類似 `http://192.168.x.x:5173` 的網址
4. 在手機瀏覽器輸入該網址即可測試
